# Flujo de trabajo: Figma a Codex/Claude Code en menos de 5 minutos

## Resumen general
El vídeo muestra un flujo de trabajo gratuito para convertir diseños de Figma en código funcional ejecutable en localhost usando Codex o Claude Code. El proceso utiliza **Magic Path** como intermediario ("canvas") para importar frames de Figma, generar variantes de estilo y crear páginas completas con comprensión de contexto (ej. páginas de blog detalladas). El desarrollador demuestra cómo autenticar la herramienta, importar diseños, iterar variantes, solicitar generación de código a la IA y ejecutar el resultado localmente. El flujo es aplicable a webs, landing pages, web apps, apps móviles (iOS/Xcode) y apps Electron.

## Ideas principales
- **Puente Figma → Código:** Magic Path actúa como capa intermedia que importa selecciones de Figma (vía link de frame) y las hace accesibles a agentes de IA (Codex/Claude Code).
- **Instalación vía "Skill":** Se instala una *skill* copiando un comando desde la documentación de Magic Path (External Agents) y pegándolo en el agente (Codex/Claude Code), seguido de reinicio y autenticación (`Authenticate Me`).
- **Generación de variantes:** Magic Path permite duplicar y remixar diseños para crear variantes de estilo automáticas (ej. neo-brutalismo, cambios de layout/tags).
- **Contexto superior en Magic Path:** Al pedir una página de blog, Magic Path genera diseños completos y contextualizados (listas de ingredientes, citas, artículos relacionados), mientras que el agente solo genera una estructura básica sin instrucciones complejas.
- **Iteración humano-IA:** El flujo permite seleccionar un componente en Magic Path → pedir al agente que lo implemente/reemplace en el código → ejecutar en localhost → repetir.
- **Multiplataforma:** El mismo flujo funciona para proyectos web (localhost), apps móviles (abriendo el resultado en Xcode para iOS) y apps Electron.

## Conceptos, herramientas y datos relevantes
- **Herramientas principales:** Figma, Codex, Claude Code, Magic Path (magicpath.ai), Xcode.
- **Mecanismo clave:** *Skill* de Magic Path para External Agents (instalable via comando en el agente).
- **Comandos clave:** `Authenticate Me` (inicio de sesión), `copy link to selection` (Figma, Cmd+L).
- **Capacidades Magic Path:** Importación de frames, generación de variantes de estilo, diseño contextual de páginas completas, exportación a código nativo (iOS/Swift).
- **Limitación mencionada:** Cuenta gratuita con límite de sitios/proyectos (número exacto no especificado).

## Conclusión breve
Magic Path cierra la brecha entre diseño visual y generación de código contextualizado, permitiendo a desarrolladores usar agentes de IA (Codex/Claude Code) para implementar interfaces completas y responsivas —incluyendo navegación y páginas internas— a partir de frames de Figma, con un flujo rápido, gratuito para empezar y extensible a móvil y desktop.
